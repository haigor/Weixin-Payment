# Weixin-Payment
==========================

参考文档https://pay.weixin.qq.com/wiki/doc/api/jsapi.php?chapter=9_1

项目地址 https://github.com/FishOfKing/Weixin-Payment
##Features

- 支持公众号微信统一下单，订单查询，关闭订单

##Install

via npm:

```bash
$ npm install weixin-payment
```
